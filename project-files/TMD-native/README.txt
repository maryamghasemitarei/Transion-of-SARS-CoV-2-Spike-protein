This folder containes:
1- the folder named "native_down_to_up" contains nine subfolders, "replica_1" to "replica_9," each corresponding to the targeted MD simulations of the transition from the native down state to the up state of the SARS-CoV-2 spike protein.
2- the folder named "native_up_to_down" contains nine zipped files, "replica_1" to "replica_9," each corresponding to the targeted MD simulations of the transition from the native up state to the down state of the SARS-CoV-2 spike protein.

Any sub-folders has its own README file.
