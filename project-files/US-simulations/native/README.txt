This folder containes:
1- umbrella sampling (US) parameters for GROMACS 2020 software (umbrella.mdp file)
2- the folder named "toppar" contains the bonded and non-bonded parameters of atoms in charmm36 force field.
3- the zipped files named "replica_1", "replica_2", and "replica_3" contains all input files for the US simulations and the US windows obtaining from targeted MD simulation  during the transition of the native SARS-CoV-2 Spike protein from the down to the up configuration for three different systems.

Any sub-folders has its own README file.
